# settings.json チェックリスト

## スコープ

settings.jsonの構文、permissions設定、hooks、plugins、カスタム設定を検証。

---

## Critical

### 構文・スキーマ
- [ ] 有効なJSONか（構文エラーがないか）
- [ ] `$schema`が指定されているか

---

## High

### permissions
- [ ] `deny`/`allow`が適切に設定されているか
- [ ] 機密ファイル（`.env`, `*.key`, `*credentials*`等）が`deny`に含まれているか
- [ ] 広すぎる許可（`Bash(*)`等）がないか
- [ ] スクリプト実行許可が必要最小限か

### hooks
- [ ] hooks設定がある場合、スクリプトパスが有効か
- [ ] 適切なタイミング（PostToolUse, Stop等）で実行されているか
- [ ] hooksスクリプトが存在するか

---

## Medium

### plugins
- [ ] `enabledPlugins`の設定が意図的か
- [ ] 不要なプラグインが有効になっていないか
- [ ] プラグインの設定が適切か

### カスタム設定
- [ ] `statusLine`設定のスクリプトが存在するか
- [ ] `fileSuggestion`設定のスクリプトが存在するか
- [ ] カスタム設定のパスが正しいか

---

## Low

### env
- [ ] 環境変数の設定が適切か
- [ ] 機密情報が含まれていないか
- [ ] 環境変数名が明確か

### 整理
- [ ] 設定が論理的にグループ化されているか
- [ ] コメントで説明が必要な複雑な設定がないか
