# Agents チェックリスト

## スコープ

カスタムAgent定義のfrontmatter、ロール定義、プロンプト設計、ツール権限を検証。

---

## Critical

### frontmatter必須項目
- [ ] `name` が存在するか
- [ ] `description` が存在するか
- [ ] `tools` が定義されているか

---

## High

### ロール定義の明確さ
- [ ] 専門性が明確に定義されているか（「あなたは○○専門の□□です」形式）
- [ ] 原則・行動指針が列挙されているか
- [ ] ロールに必要な専門知識が記載されているか

### 成功基準
- [ ] 具体的な成功基準があるか
- [ ] チェックリスト形式で提示されているか
- [ ] 完了判断が明確か

### 出力形式
- [ ] 出力形式のテンプレートが提供されているか
- [ ] 結果統合を考慮した構造化出力か
- [ ] 一貫性のある出力形式か

---

## Medium

### ツール権限の最小化
- [ ] 必要最小限のツールのみ許可しているか
- [ ] 読み取り専用タスクに書き込みツールが含まれていないか
- [ ] 危険なコマンドが`allowedCommands`に含まれていないか

### プロンプト構造
- [ ] ロール → 原則 → タスク → 成功基準 → 出力の順で構成されているか
- [ ] セクション分けが明確か
- [ ] 必要な情報が過不足なく含まれているか

### コンテキスト分離
- [ ] 独立して実行可能な設計か
- [ ] 他のAgentとの依存関係が明確か
- [ ] 並列実行時のファイル競合を考慮しているか

---

## Low

### 命名
- [ ] Agentの役割が名前から推測できるか
- [ ] 一貫した命名規則を使用しているか

### 冗長性
- [ ] 既存の組み込みAgent（Explore, Plan等）で代替できないか
- [ ] 説明より具体例が優先されているか
