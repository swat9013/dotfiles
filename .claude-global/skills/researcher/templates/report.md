## 記入ガイド

### 比較表の評価軸例

| 用途 | 評価軸の例 |
|-----|----------|
| ライブラリ選定 | バンドルサイズ、型サポート、学習コスト、コミュニティ、メンテナンス状況 |
| アーキテクチャ | スケーラビリティ、複雑度、テスト容易性、移行コスト |
| インフラ | コスト、運用負荷、可用性、ベンダーロックイン |

### 評価の表記

| 記号 | 意味 |
|-----|------|
| ++ | 非常に良い |
| + | 良い |
| - | やや劣る |
| -- | 劣る |
| ? | 未調査/不明 |
