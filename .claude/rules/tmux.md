# Tmux設定アーキテクチャ

## 基本設定

### Prefix キー

```
prefix: C-z (Ctrl+z)
```

**理由**: デフォルトの `C-b` より押しやすく、他のキーバインドと競合しにくい

## ペイン操作

| キー | 機能 |
|------|------|
| prefix + \| | 垂直分割 (左右に分割) |
| prefix + - | 水平分割 (上下に分割) |

分割・新規ウィンドウは `#{pane_current_path}` で現在のパスを引き継ぐ

## ペイン・ウィンドウ移動

| キー | 機能 |
|------|------|
| Ctrl + 矢印キー | ペイン移動（prefix不要） |
| Shift + 矢印キー | ウィンドウ移動（prefix不要） |

## Vi キーバインド (コピーモード)

- `v`: 選択開始
- `y`: `pbcopy` へコピー
- マウスドラッグ: コピー → `pbcopy`

## ステータスバー

カスタム軽量ステータスバー（上部配置）:
- 左: セッション名（緑背景）
- 中央: ウィンドウリスト（アクティブ=青背景）
- 右: 日付 時刻

## プラグイン（TPM）

| プラグイン | 用途 |
|------------|------|
| tmux-sensible | 合理的デフォルト |
| tmux-resurrect | セッション保存/復元 |
| tmux-continuum | 自動保存（15分間隔）+ 自動復元 |
| tmux-yank | クリップボード連携 |

**操作**: `prefix + I`=インストール, `prefix + Ctrl-s`=保存, `prefix + Ctrl-r`=復元

## IDE レイアウト (lib/ide.sh)

`lib/ide.sh` で以下のレイアウトを自動作成:

```
┌──────────────┬─────────┐
│              │         │
│              │  logs   │
│   editor     │         │
│              ├─────────┤
│              │         │
│              │  shell  │
└──────────────┴─────────┘
```

エイリアス `ide` で起動可能。
