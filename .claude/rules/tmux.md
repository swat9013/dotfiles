# Tmux設定アーキテクチャ

## 基本設定

### Prefix キー

```
prefix: C-z (Ctrl+z)
```

**理由**: デフォルトの `C-b` より押しやすく、他のキーバインドと競合しにくい

## ペイン操作

| キー | 機能 |
|------|------|
| prefix + \| | 垂直分割 (左右に分割) |
| prefix + - | 水平分割 (上下に分割) |

**覚え方**: `|` は縦線、`-` は横線を表す視覚的な表現

## Vi キーバインド (コピーモード)

コピーモードで Vi風のキーバインドを使用:

- `h`, `j`, `k`, `l`: カーソル移動
- `/`: 検索
- `v`: 選択開始
- `y`: ヤンク（コピー）

### macOS クリップボード統合

`reattach-to-user-namespace` を使用して、tmux内でのコピーをmacOSクリップボードと連携。

## Powerline統合

ステータスラインにPowerlineを使用:
- セッション名
- ウィンドウ一覧
- ペイン情報
- システム情報（CPU、メモリなど）

## IDE レイアウト (lib/ide.sh)

`lib/ide.sh` で以下のレイアウトを自動作成:

```
┌──────────────┬─────────┐
│              │         │
│              │  logs   │
│   editor     │         │
│              ├─────────┤
│              │         │
│              │  shell  │
└──────────────┴─────────┘
```

エイリアス `ide` で起動可能。
