# Zshキーバインド設計

## キーバインド一覧

| キー | 機能 | 説明 |
|------|------|------|
| Ctrl+r | fzy履歴検索 | コマンド履歴をインタラクティブに検索 |
| Ctrl+s | fzyディレクトリナビゲーション | ディレクトリをインタラクティブに選択・移動 |
| Ctrl+^ | 一つ上のディレクトリへ | `cd ..` の短縮 |
| Ctrl+m | 入力なし→ls+git status | 空行でEnter時に自動実行 |
| Ctrl+v | VSCode起動 | カレントディレクトリをVSCodeで開く |

## fzy統合

**fzy**: 軽量で高速なインタラクティブフィルタ（pecoの後継）

### Ctrl+r (履歴検索)

コマンド履歴を逆順で表示し、fzyでフィルタリング。選択したコマンドをコマンドラインに挿入。

### Ctrl+s (ディレクトリナビゲーション)

カレントディレクトリ配下のディレクトリを検索し、選択したディレクトリに移動。

## 設計方針

- **インタラクティブ選択**: キーボード操作で効率的に選択
- **軽量**: fzyは起動が高速で、大量のデータでもスムーズ
- **Unix哲学**: 小さく単機能なツールの組み合わせ
